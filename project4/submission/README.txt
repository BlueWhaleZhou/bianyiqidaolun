1 The source code Shihuan_Shao_Proj4.c and head file syntax.h should be put into the same file folder. 

2 Compiling instructions:
gcc Shihuan_Shao_Proj4.c

This will generate a file called a.out in the general machine.

3 Executing instructions:
./a.out <filein> fileout
Assuming the input file name is test1.txt, and the output file name is output1.txt, the instruction will be 

./a.out <test1.txt> output1.txt
